# About: Quarto Book Template

Book template for DOCX to Quarto.

Version: Minimal citation config, no bibliography.

## Configuration

### Citations and bibliographies

This template version does not use any additional citation processing by Quarto, beyond its default method.

Pandoc is the Quarto default for citations.

Pandoc will attempt to process citations unless turned off.

Optional programmes Quarto uses for rendering outputs: Pandoc, Citation Style Language (CSL) and citeproc, LaTeX: using BibLaTeX or natbib.

You can have multiple bibliographies, one per file, or/and one for a book.

See: https://quarto.org/docs/authoring/citations.html
